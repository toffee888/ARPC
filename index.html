Tutorial (Handphone/Tablet):

Gerakan hp ke kanan dan kiri untuk mencari piala

Jikalau piala tidak ditemukan, mohon refresh

Setelah piala ditemukan:

Gunakan satu jari untuk memutar piala

Gunakan dua jari untuk membesar/mengecilkan objek (Pinch/Zoom)

Gunakan tiga jari untuk mengerakan piala keatas, kebawah dan kesamping (Drag)

Tekan tombol screenshot untuk mengambil gambar



Tutorial (PC):

Click kiri dan gerakan mouse untuk memutar piala

Click kiri dan kanan untuk mengerakan piala

Gunakan mouse wheel untuk membesar kecilkan piala

Tekan tombol screenshot untuk mengambil gambar
